# react-native-inappstory-sdk

Wrapper for InAppStory

## Installation

```sh
npm install react-native-inappstory-sdk
```

## Usage

```js
import { InappstorySdkView } from "react-native-inappstory-sdk";

// ...

<InappstorySdkView color="tomato" />
```
